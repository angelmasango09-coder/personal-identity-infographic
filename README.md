# personal-identity-infographic
Personal identity infographic and logomark for CMPG212 assignment

This repository contains my personal identity infographic developed for the CMPG212 assignment.

The infographic represents my identity through visual design and a personal logomark.

## Repository Contents
- Infographic
- Personal logomark
 

## Personal Focus
My identity system represents:
- Growth and continuous development
- Authenticity and individuality
- My journey as a student and future professional

## Logomark Meaning
The incomplete circle symbolises continuous growth and progress.  
The rough edges and uneven shading represent individuality and the imperfection.
